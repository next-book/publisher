publisher test

What is this?

This is an in-browser test of next-book's publisher — it breaks element's content into „ideas“ and „chunks“ that may be then used for enhanced display or interactivity within a larger bulk of text.

In following examples, only elements with class map are mapped and there's some border-bottom and :hover CSS apllied to them to higlight the mapped structure. Both mapped and plain elements should otherwise look the same.

See API docs for a rough idea of how publisher works. (Still battling with documentation of ES6 modules that do not use classes internally.)
