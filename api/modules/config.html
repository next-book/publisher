Module config

Config module

The config is being created in following stages:

  1. First in prepConfig, the custom book options in the shape of PartialConfig are loaded.
  2. Preview feature settings in shape of Preview are being created by appling loaded custom options onto previewDefaults. The preview feature settings are added to the loaded config, together making the shape of PartialConfigWithPreview.
  3. Since config is guaranteed to have preview settings, it is then used to override {@link configDefaults}.
